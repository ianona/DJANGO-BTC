How do the values of other cryptocurrencies behave similarly or differently from Bitcoin?

Cryptocurrency is a type of digital currency regulated by the use of encryption techniques. Although Bitcoin is probably the most well known cryptocurrency, there exists other altcoins that serve as alternatives to Bitcoin. This question aims to identify any existing relationship between these altcoins and Bitcoin.

Initial Analysis Additional Statistics Code

Preparing the data

To begin the analysis, we first collected data from 5 different datasets which described the prices of Bitcoin, Ethereum, Litecoin, Iota, and Monero. We then cleaned these datasets and solved for a new usable column to represent the OHLC Average of the currency. We then merged the dataframes into one to serve as our working dataframe.

Plotting the data

We first plotted the dataframe into a singular line graph for us to compare the trends between the prices of these cryptocurrencies. The first thing we noticed was the exponentially larger price and increase of Bitcoin over the same period of time these coins experienced stagnant increases. However, the researches wanted to look for relationships beyond the magnitude of the price, which led them to perform additional statistical testing.

Pair Plot with Linear Regression

We first performed a pair plot test with linear regression using the plotting funcions of seaborn. The results of the test appeared to be similar across all pairs, all of which indicated a positive linear relationship between each of the currencies.

Multiple Pearson's Correlation

We then performed a Multiple R test on the dataframe to quantify the coefficient of correlation between each pair. Similar to the pair plot and linear regression testing, the R values found were close to positive one (1), indicating very strong positive linear relationships between the variables. These findings helped the researchers conclude that although the price of Bitcoin is still very much ahead of other altcoins, they all experience increase which can be due to other factors like the overall popularity of cryptocurrency as it became more known over time.

for x in df:

    dates=[]
