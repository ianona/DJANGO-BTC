Does popularity in terms of media exposure and amount of investors contribute to the behavior of the value of Bitcoin?

One of the more social media platforms such as twitter, facebook, instagram and etc. have people talking about numerous different topics and numerous different times. There is a function in twitter that allows you to see the most trending topics people are tweeting about. Because of twitter's api and the api of these popular social media platforms and its ability to track the number of #(hashtags) of a certain topic, the number of mentions of the word "Bitcoin" can be tracked and monitored to see how much people are actually tweeting about it. This will show how much bitcoin exposure there is in across all popular social media platforms.

Model Analysis Code

Model Assumptions

The Data downloaded must be checked if their models are safe to make assumptions. These conditions are satisfied if:

  • The data is NORMALLY DISTRIBUTED
  • There are CONTINUOUS numeric values
  • Variables are LINEARLY RELATED
  • Variables are INDEPENDENT of each other

Because our p-value is 0.23719675904068735, we accept the null hypothesis which means that these two variables are independent which follows the model assumptions

Does NOT defy assumptions!

  • The upper left graph and lower right graph somewhat has a bell curve
  • There are linear relationships between the lower left and the upper right graph
  • There are no continious values in any of the graphs
  • Variables are independent of each other via chi-squared test

Pearson's Correlation

We can use Pearson's correlation since we do not defy model assumptions. Using Scipy to check for Pearson's correlation after confirming a correlation, we find that the model has a Pearson's coefficient of -0.011694552922683999
